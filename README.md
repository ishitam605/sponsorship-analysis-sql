# 📊 Sponsorship Revenue & Client Analysis (SQL Project)

## 📌 Overview
This project focuses on analyzing sponsorship data to identify key revenue drivers, high-value clients, and business insights. The analysis is performed entirely using SQL.


## 📂 Dataset
- Contains 50+ client records across multiple industries
- Includes:
  - Sponsorship amount
  - Event type
  - Engagement score
  - Renewal status
- Covers multiple years (2023–2024)

---

## 🛠️ Tools & Technologies
- SQL (MySQL)

---

## 🔍 Key Analysis Performed

### 1. Data Cleaning
- Checked for missing values
- Identified duplicate records
- Standardized categorical fields

### 2. Revenue Analysis
- Total sponsorship revenue
- Revenue by industry
- Revenue by event type
- Year-wise revenue trends

### 3. Client Analysis
- Identified top revenue-generating clients
- Ranked clients using SQL window functions
- Segmented clients into High, Medium, and Low value

### 4. Engagement & Retention
- Analyzed client engagement levels
- Studied renewal patterns
- Correlated engagement with retention

---

## 📈 Key Insights
- Tech industry generates the highest sponsorship revenue  
- Conferences are the most profitable event type  
- High-value clients contribute the majority of total revenue  
- Clients with higher engagement scores show better retention rates  
- Revenue increased in 2024 compared to 2023  

---

## 📊 SQL Concepts Used
- Aggregate functions (SUM, COUNT, AVG)
- GROUP BY and ORDER BY
- CASE statements for segmentation
- Window functions (RANK)
- Data cleaning techniques
